# موقع الإطلالة العربية

## نظرة عامة

موقع إلكتروني احترافي لشركة الإطلالة العربية للتدريب والاستشارات، تقدم حلولاً تدريبية واستشارية متكاملة للمؤسسات والأفراد في المنطقة العربية.

## المميزات

- تصميم احترافي متجاوب مع جميع الأجهزة
- واجهة مستخدم باللغة العربية (RTL)
- صفحات متعددة تشمل:
  - الصفحة الرئيسية مع جميع الأقسام
  - صفحة الدورات التدريبية
  - نماذج التواصل
- تأثيرات بصرية متقدمة وانيميشن
- تحسين محركات البحث (SEO)
- روابط وسائل التواصل الاجتماعي
- قسم العملاء والشركاء
- عرض للدورات التدريبية المتخصصة

## المحتوى

### الخدمات المقدمة
- برامج تدريبية متخصصة
- استشارات إدارية واستراتيجية
- تطوير العمليات والجودة

### الدورات التدريبية
- تدريب المدربين (TOT)
- تعليم القواعد النحوية لغير الناطقين بالعربية
- ندوات الأمن السيبراني
- التحليل الإحصائي SPSS
- التحول الرقمي

### الفريق
- د. أميرة عبد الحي - المؤسسة والمديرة التنفيذية
- د. محمد عبدالمنعم عبدالحي - أستاذ اللغة العربية
- د. منال البلقاسي - مستشارة ومدربة معتمدة
- د. عثمان خليل - خبير علم النفس التربوي

## التقنيات المستخدمة

- HTML5
- CSS3 مع متغيرات مخصصة
- JavaScript متقدم
- Font Awesome للأيقونات
- خطوط Google (Tajawal)
- تأثيرات Particles.js
